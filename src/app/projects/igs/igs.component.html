IGS – Ital Gas Storage

Web application for storing and distribution of gas in Italy
  • Developed microservices for adding new app features (file storage, user creation, administration, gas storage data microservices, tickets formaintaining app...)
  • Development of RESTful web services(Spring Boot) for various data retrieving and manipulation requested by front-end side which I’ve done in Angular 5
  • Implementation of Angular smart tables for data representation, adding clone, editing data and graph filtering by time functionalities
  • Designed and implemented various chart representation of data using AmCharts
  • Technologies: TypeScript(Angular 5), Java(Spring Boot, Spring Data), MSSQL, JHipster, Git
